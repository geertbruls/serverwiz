# Non-functional requirements

Non-functional requirement attributes are based on the [ISO 25010 norm](https://nl.wikipedia.org/wiki/ISO_25010).

| ID   | Requirement                                                                      | Attribute   | Priority  |
| ---- | -------------------------------------------------------------------------------- | ----------- | --------- |
| NF01 | The artefact runs in an environment without installing dependencies/interpreters | Portability | Must have |
| NF02 | The artefact runs in a containerised environment                                 | Portability | Must have |
| NF03 | The artefact's Minecraft functionality is compliant with Mojang's TOS            | Maturity    | Must have |
